-- [Como utilizar a pasta de scripts da Peda Engine] --

> Primeiro:
Coloque seu script na pasta scripts da Peda Engine 
por SOURCE NO SEU FORK DO
GITHUB E NÃO PELO APK EDITOR!!

> Segundo:
Pós isso faça um script na sua música com o seguinte
codigo:

function onCreate()
if (songName == 'suamusica') then
addLuaScript('data/scripts/seuscript')
end
end

> Terceiro:
Pronto agora você tem um script funcionando!
se você tem mais de um apenas copie o codígo
e edite o nome do script e da musica

a pasta vem com 4 scripts templates pra você
utilizar

-- [How to use the Peda Engine script folder] --

> First:
Place your script in the Peda Engine scripts folder
BY SOURCE IN GITHUB ON YOUR OWN FORK
NOT IN APK EDITOR!!

> Second:
After that put a script in your song with the following code:

function onCreate()
if (songName == 'your song') then
addLuaScript('data/scripts/yourscript')
end
end

> Third:
You now have a working script!
if you have more than one just copy the code
and edit the name of the script and the song

the folder comes with 3 script templates for you
use
